Gmail
Connect via Google OAuth

Access your email inbox, view recent messages, and get unread counts.

First-time setup: You need Google OAuth credentials. How to get them?

How to Get Gmail OAuth Credentials

Step 1: Go to Google Cloud Console

Step 2: Create a new project (or select an existing one)

Step 3: Enable the Gmail API

  • Go to "APIs & Services" ‚Üí "Library"
  • Search for "Gmail API"
  • Click on it and press "Enable"

Step 4: Configure OAuth Consent Screen

  • Go to "APIs & Services" ‚Üí "OAuth consent screen"
  • Choose "External" user type
  • Fill in app name (e.g., "MyBrain")
  • Add your email as test user

Step 5: Create OAuth Credentials

  • Go to "APIs & Services" ‚Üí "Credentials"
  • Click "Create Credentials" ‚Üí "OAuth client ID"
  • Application type: Desktop app
  • Name it "MyBrain Desktop"
  • Click "Create"

Step 6: Copy your credentials

  • Copy the Client ID (ends with .apps.googleusercontent.com)
  • Copy the Client Secret (starts with GOCSPX-)
  • Paste them in the fields above

Note: Your credentials are stored securely on your local machine and are never sent anywhere except to Google for authentication.
